Lovable vs v0 vs Bolt vs Replit：免费额度、价格、场景一次看懂

## 1. 什么是 Vibe Coding？

一句话：用嘴写代码。  
由 Andrej Karpathy 提出，核心就是：  
**自然语言描述需求 → AI 生成可运行应用 → 一键部署上线**。  
今天我们把最热门的 4 个云端 Vibe Coding 平台拉出来遛一遛，并用同一套 Prompt 做实测，再补上最新的免费额度与价格，让你看完就能直接开干。
