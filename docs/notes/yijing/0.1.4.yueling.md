> 来源于《古筮真诠》

月令，即我们在卦算中，日月组合里头所见的`月支`。

月令作为日月组合的构成之一，其无论在吉凶判断层面，或在应期细节层面，影响力与信息功能都是非常重要的。

古人对此则更为重视，明朝易卜经典《黄金策》有曰：“月将乃万卜之提纲。掌一月之权，司三旬之令”。`月将`就是指`月建`，又称为`月令`，意思是说，`月令`在一月上中下旬三十天内，**当权得令**，在此期间它的地位是至高无上的。

我们知道一年有十二月令，按中国古代传统，将一年中分为十二节令与十二气令，统称为二十四节气，而我们卦算中所使用的**月令**，是按照其中的十二节令划分的，规划如下：正月为寅月，以立春节之日交令；立春节也是年与年之间的交令，是新一年的开始，我们每岁过年，过春节的习俗也源自于此。二月为卯月，以惊蛰节之日交令；三月为辰月，以清明节之日交令；四月为巳月，以立夏节之日交令；五月为午月，以芒种节之日交令；六月为未月，以小暑节之日交令；七月为申月，以立秋节之日交令；八月为酉月，以白露节之日交令；九月为戌月，以寒露节之日交令；十月为亥月，以立冬节之日交令；十一月为子月，以大雪节之日交令；十二月为丑月，以小寒节之日交令。这里大家有必要清晰的是，其实十二节令在一年中的日子是相对固定的，每年的误差只在一到两天之内，例如立春，每年的立春，不是在公历的2月4日，就是2月5日。由此可推算到，用于卦算的12月令时间范围，也如同公历一样，每年都是相对固定在某个时段范围内的。各月令公历时段范围大致如下表：

| 月 | 名称 | 农历时段 |
| --- | --- | --- |
|子月 |大雪| 12月7日至1月5日|

## 月令范围的认识错误

有好些初学者，误以为子丑寅卯月就是指农历的一、二、三、四月，于是一旦到了农历闰月，立刻昏头转向，搞不清楚应按哪个月令推算为准，闹出了诸如“逢农历闰六月，怎么跑出两个未月来的呢？”的笑话，**其实子丑寅卯月等用的是干支历法，与农历的各月划分并不是一回事，彼此不能等同视之，这属于基础性的错误了**。可能是受农历不是相对固定的观念影响，有一些初学者居然并不知晓十二月令时间范围的相对稳定性质，算卦总随身带着本万年历，子丑寅卯月相当于公历的哪些时日，都要到书上仔细翻查，其实没必要这样自己找自己麻烦，内行熟手的易学者，只需要记住子丑寅卯月各在公历的哪个时间范围段，跟别人分析时，便直接以公历告之，毕竟社会上绝大部分人，并不知晓子丑寅卯月各自代表公历的哪个时间段，内行人若不点明，求测人自然是你糊涂我跟着更加糊涂。


## 月令对卦象吉凶趋势的影响力

因一年有四季之分，各季节的具体划分为：寅卯辰月为木之春季，已午未月为火之夏季，申酉戌月为金之秋季，亥子丑月为水之冬季，一卦成后，卦中爻在月令里，适时得令者便得旺相，背道时令的便遭衰败，这就是月令能对卦中各爻产生旺衰影响的根本原理所在。通过本书总论篇对日月组合的单独介绍，我们已了解到月令能对卦中所有爻的旺衰产生影响，它是日月动变四大影响因素之一，月令作为日月组合的其中一个组成部分，虽然这种影响作用，在吉凶判断层面大多数情况下，并不能够与动变之爻的重要性相比拟，往往**动爻**才是事情吉凶趋势的最关键信息因素。

## 月建对爻旺衰影响的结构化细则

下表按“旺相、破伤、休囚”分类，明确定义、层次、示例与边界，便于对照应用。

| 类别 | 关系 | 定义与层次 | 示例 | 关键说明 |
| :--- | :--- | :--- | :--- | :--- |
| **旺相** | 临月建 | 爻与月建地支相同，最高层次旺相，当月金刚护体，逢克不伤 | 寅木爻逢寅月 | 月建当权，力量最强 |
| | 月建合 | 爻与月建相合（六合），高层次旺相，合旺论 | 寅木爻逢亥月（亥寅合） | 合则有力，不被轻易冲散 |
| | 月建生 | 月建五行生爻，一般层次旺相 | 寅木爻逢子月（子水生木） | 得生则旺，助力明显 |
| | 月建扶 | 爻与月建五行相同（比和），一般层次旺相 | 寅木爻逢卯月（同属木） | 同气相助，力量次于临与合 |
| **破伤** | 月建冲 | 爻与月建相冲，高层次衰相，称为月破 | 寅木爻逢申月（申寅冲） | 当月无力，出月或逢合填实可解 |
| | 月建克 | 月建五行克爻，一般层次衰相 | 寅木爻逢申月（申金克木） | 受克则衰，力量次于月破 |
| **休囚** | 爻生月 | 爻五行生月建，泄气，一般衰相 | 寅木爻逢巳月（木生火） | 耗泄自身，力量减弱 |
| | 爻克月 | 爻五行克月建，耗气，一般衰相 | 寅木爻逢未月（木克土） | 克伐月令，自身反遭耗损 |
| | 月不生扶 | 爻与月建无生扶合扶，亦无冲克，平相转休囚 | 寅木爻逢辰月（土不生木，无冲克） | 不得令，力量平庸 |
