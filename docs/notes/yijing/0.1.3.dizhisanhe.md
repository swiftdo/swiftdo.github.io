五行木火金水的十二长生中，长生、帝旺、墓库三地支的组合为地支三合。以五行木为例，亥为长生、卯为帝旺、未为墓库，亥卯未成三合木局。同理可知，寅午戌三合火局，巳酉丑三合金局，申子辰三合水局。


![](/imgs/yijing-dizhi-sanhe.jpg)

地支三合的三个地支，每两个地支之间相隔三个地支，在十二宫中正好构成等边三角形。这个等边三角形在紫微斗数中也有广泛的应用。


三合，即是明合，光明正大地合，是三个生肖的吉配，而三合局是凝聚力，是向中神凝聚的力量，合力是最强的.

三合，不但契合度佳，通常为思想、价值观、习惯等多方面非常契合，而三合生肖均相差四年，这也是所谓差4岁结婚，天作之合，便是根据「三合」而来。
