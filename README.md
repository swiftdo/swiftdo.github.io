用人工智能生成文章：

提示词：

```
请写一篇文章关于 特征工程 的文章，文件放到 docs/ai/ml 目录.

具体步骤:
1. 首先，你需要明确你想要写的文章的主题。
2. 然后，你需要收集相关的资料，包括书籍、文章、论文等。
3. 接着，你需要对这些资料进行整理，将它们转化为你想要的格式。
4. 最后，你需要将整理好的资料写成一篇文章。
5. 文件名放到sidebarConf对应位置


文章内容要求：

1. front format用这种格式：

---
title: "监督学习和无监督学习"
date: 2025-02-03
tags:
  - ai
  - ml
  - 机器学习
sitemap:
  exclude: false
  changefreq: hourly
---

2. 文章内容需要通熟易懂。
3. 文章内容需要有实际应用的例子。
4. 如是比较形内容，请用表格来展示异同、优缺点等。
```
