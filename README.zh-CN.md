# old-bird

网站使用 [vuepress](https://vuepress.vuejs.org/) 和 [vuepress-theme-plume](https://github.com/pengzhanbo/vuepress-theme-plume) 构建生成。

## Install

```sh
yarn
```

## Usage

```sh
# 启动开发服务
yarn docs:dev
# 构建生产包
yarn docs:build
# 本地预览生产服务
yarn docs:preview
# update vuepress and theme
yarn vp-update
```
