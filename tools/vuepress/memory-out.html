    # 解决方案

    在 package.json 修改配置：

    // package.json 配置
"scripts": {
    "docs:build": "node --max_old_space_size=4096 ./node_modules/vuepress/cli.js build docs",
},

    修改了 docs:build 的执行语句，指定了内存为 4096。
